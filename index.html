Twitch.tv Clone

A clone of twitch.tv, with full live streaming capability

Built using React / Redux and Ruby on Rails with PostgreSQL database, through ActiveRecord. Configured a Centos/Linux Digital Ocean server to accept RTMP streams, and accept HTTP GET requests. Used the HLS.js library to display a video feed corresponding with the user's index file, from the user's RTMP stream.

Utilized AWS and S3 cloud services, with Rails ActiveStorage, for hosting user profile and streaming category pictures.

Implemented a responsive layout, by leveraging Flexbox and the React-Redux life-cycle in order to re-render and dynamically resize key React Components.

Global Life

Data Visualization of Countries

Navigable globe using Three.js, where data about each country is displayed using D3.js.

Upon hovering over each country, an additional 'texture' layer (represented as a canvas where the path of the stroke is the outline of the country) is rendered. Note: Higher amounts of segments lead to more processing power required, but more accurate outlines.

Upon clicking a country, data from the World Happiness Index of 2019 is displayed in the top left as a bar chart, representing the country's rating relative to the other 156 countries in categories such as Corruption, Negative Affect, and Life Expectancy of their respective citizens. Additional data pulled from multiple sources.
