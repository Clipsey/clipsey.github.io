All About Me

The Software Engineer

Hi. I'm Ryan Sabik. I have a Computeer Engineering degree from the University of California - Riverside. I've been employed for an Army research base, then called TARDEC, as a Computer Engineer on the Robotics Engineering Team, where I wrote Python code for various tasks, like automating controller interactions with the robotic platform by spoofing JAUS traffic observed over Wireshark.

After, I worked as an Embedded Software Engineer for General Motors on the Voice Recognition and Voice assistant teams, where I wrote C++ and Java code on the Android Platform. I also wrote automation frameworks in Python and Bash Scripts to create artificial voice input and touch interactions with the infotainment system.

Most recently, I've been sharpening my FullStack Web Development skills, where I've been building projects in React, Redux, Node.js / Express, Ruby on Rails, using MongoDB, PostgreSQL, WebSockets (socket.io), D3.js, Three.js, and Webpack.
